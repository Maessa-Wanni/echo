# Dashboard

Het dashboard is de hoofdinterface voor het analyseren van je gespreksgegevens. Het biedt een breed scala aan tools en functies om je te helpen je data te verkennen, analyseren en begrijpen.

## Overzicht

De startpagina is de eerste pagina die je ziet wanneer je inlogt. Het toont al je projecten. Selecteer een project om te beginnen met analyseren. Maak een nieuw project aan door op de "Aanmaken" knop te klikken.

Dashboard-instellingen zijn te vinden in de rechterbovenhoek. Hier kun je een probleem melden en de taal van je dashboard wijzigen.

Wanneer je een project selecteert, zie je de volgende secties:

- **Project Instellingen**: Wijzig de naam van je project en voeg context toe die ECHO kan helpen om je project en de doelstellingen beter te begrijpen. Hier kun je ook al je transcripten exporteren en je project verwijderen.
- **Portal Editor**: Hier kun je de deelnemersportal van het project maken en bewerken. Wijzig de taal, vraag om een naam of houd het anoniem, stel het type tutorial in, maak een lijst met tags voor de deelnemers om te selecteren, en meer.
- **Vragen**: Klik hierop om te beginnen met chatten met ECHO. Je chatgeschiedenis wordt opgeslagen in de sectie "Chats".
- **Bibliotheek**: Klik hierop om naar de inzichten [bibliotheek](library) te navigeren.
- **[#] Chats**: Een verzameling van je chats met ECHO.
- **[#] Gesprekken**: Een verzameling van gesprekken en hun transcripten.
